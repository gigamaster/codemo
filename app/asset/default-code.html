wooorm/starry-night

Syntax highlighting, like GitHub

Example: using starry-night on the client You don’t have to do preprocess things on a server. Particularly, when you are not using Node.js or so. Or, when you have a lot of often changing content (likely markdown), such as on a page of comments. In those cases, you can run starry-night in the browser. Here is an example. It also uses hast-util-to-dom, which is a light way to turn the AST into DOM nodes.

https://github.com/wooorm/starry-night
